## Install

```bash
git clone https://github.com/Dauno/shopping-cart-back.git
$ cd /shopping-cart-back
$ npm i
```

## Run

http://localhost:3001/api/product

http://localhost:3001/api/discount

local:
```bash
$ npm run start
```

docker:
```bash
$ docker-compose up --build
```

## Test
```bash
$ npm run test
```
